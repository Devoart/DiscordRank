# DiscordRank

(https://jitpack.io/v/None-Of-Ever/DiscordRank.svg)

### Dependency
```gradle
	allprojects {
		repositories {
			maven { url 'https://jitpack.io' }
		}
	}
```

```gradle
		implementation 'com.github.None-Of-Ever:DiscordRank:1.0.0'
```
